MLRISC

A framework for retargetable and optimizing compiler back ends

Lal George
Allen Leung
Bell Labs New York University

MLRISC is a customizable optimizing back-end written in Standard ML of New Jersey . The MLRISC system is currently undergoing substantial design changes and development. This document is a general overview of the architecture of the new system and describes some of the new changes to the core.

There is also a separately maintained page introducing the core MLRISC system, which is available here as the MLRISC quick tour. The pages here give a more detailed look of the MLRISC system. Please follow the links on the left.
