Support for C and C--

MLRISC has been enhanced to support languages such as C and C--. Specific enhancements include:
  • Integer and floating point operations of various widths have been included
  • Loads and stores of various widths, plus the ability to sign-extend and zero-extend loads have been included
  • Conversions between floating point and integer types are included
In addition, C-- also requires the ability to specify unaligned loads and stores. These have also been added.

The new MLTREE language has been extended to allow all these to be specified.
